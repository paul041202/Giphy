# Giphy GIF Search App Setup

## Quick Start

1. Get your Giphy API key from https://developers.giphy.com/
2. Copy `.env.example` to `.env`
3. Replace `your_giphy_api_key_here` with your actual API key in `.env`
4. Run `npm start`

## Features
- Search for GIFs using Giphy API
- Responsive grid layout
- Loading states and error handling
- Clean, modern UI
